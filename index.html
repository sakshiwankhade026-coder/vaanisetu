How it works

From voice note to scheme list in 60 seconds

No app. No English. No literacy required. Just WhatsApp — which 500 million Indians already use.

1
User sends a voice note
Rekha opens WhatsApp and sends a voice message in Hindi: "Mera naam Rekha hai, Maharashtra mein rehti hoon, do acre zameen hai, vidhwa hoon…"
WhatsApp Cloud API
2
Sarvam AI transcribes in her language
The audio is transcribed using Sarvam AI's Indic-native speech model — supports Hindi, Marathi, Tamil, Telugu, Bengali, and 6 more languages.
Sarvam AI — Indic STT
3
LLM matches schemes with confidence scoring
GPT-4o matches her profile against 24 central + state schemes. Each match returns a confidence score — borderline cases are flagged for human review, not given wrong advice.
GPT-4o · Scheme DB · Confidence Scoring
4
Voice response + document checklist
Rekha hears a clear audio reply naming her schemes, benefit amounts, and exactly what documents to bring. A document checklist card is sent alongside — she can show it at the CSC.
Sarvam AI — Indic TTS
5
Nearest CSC pin + follow-up reminder
VaaniSetu sends a Google Maps pin to the nearest Common Service Centre. 7 days later, it follows up: "Rekha ji, did you visit the CSC? Reply YES/NO."
Google Maps API · Follow-up Loop

Last-mile delivery

VaaniSetu sends the exact location

After scheme matching, the bot automatically sends a Google Maps pin to the nearest Common Service Centre — the last step is getting there.

5.5 Lakh+
Common Service Centres across India
Every CSC can process government scheme applications for free. VaaniSetu navigates users to the one closest to them.
IVRS fallback
For users without internet
Sarvam AI supports IVR (phone call) mode — users with only 2G or basic phones can call a number and get the same service. No WhatsApp required.

Technology

Built on real, production-ready infrastructure

Every component is available today, at low or zero cost, and already deployed at Indian scale.

🗣️
Sarvam AI
Indic-native speech-to-text and text-to-speech. Supports 10+ Indian languages including Hindi, Marathi, Tamil, Telugu, Bengali.
Core AI
💬
WhatsApp Cloud API
Meta's official free API. Handles voice notes, text, images, and location pins. 500M Indian users already on platform.
